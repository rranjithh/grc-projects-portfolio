📌 Overview

This repository contains hands-on GRC (Governance, Risk & Compliance) portfolio projects simulating real-world SaaS environments. The projects demonstrate practical implementation of ISO 27001:2022, risk management, incident response, and vendor risk frameworks.

📂 Projects Included
🔹 1. Risk Assessment & Risk Register — BrightPay Fintech
Performed risk assessment using a 5×5 likelihood × impact matrix
Built an Excel-based risk register with ownership, review cadence, and escalation
Developed risk heatmaps (inherent & residual) for executive reporting
Implemented controls (MFA, RBAC, encryption, WAF)

📊 Outcome: ~60% reduction in high-risk exposure

🔹 2. ISO 27001:2022 ISMS Implementation — CloudVault SaaS
Conducted gap analysis across all 93 Annex A controls
Developed Statement of Applicability (SoA)
Created remediation roadmap (30/60/90 days)
Drafted policies: Access Control, Incident Response, HR Security

📊 Outcome: ~70% reduction in control gaps

🔹 3. Incident Response Plan — NovaPay SaaS
Designed IR lifecycle aligned to NIST SP 800-61 & ISO 27035
Built severity matrix (P1–P4), RACI, and playbooks
Conducted ransomware tabletop simulation

📊 Outcome: ~50% improvement in MTTR

🔹 4. Vendor Risk Assessment — TrustLine SaaS
Developed 20-question vendor security questionnaire
Built 0–100 scoring model with risk tiering
Recommended SLA clauses & DPA controls (DPDP Act 2023)

📊 Outcome: Scalable vendor risk framework implemented
